# Sandy Enterprise DataOps Accelerator

> **Metadata drives everything. AI reasons. Python executes.**

A metadata-driven enterprise data engineering and test automation platform built on **Snowflake, dbt, Python, and AI**. This repository is the single source of truth — it houses all code, data models, configurations, documentation, and AI agent assets across every phase of the platform.
